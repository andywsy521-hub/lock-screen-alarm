# 定时活动锁屏闹铃

一个简单的桌面应用程序，用于定时提醒休息并强制锁屏。

## 功能特点

- 设置自定义定时时间（分钟）
- 时间到达后全屏锁屏5分钟
- 锁屏期间可手动解锁
- 自动重新开始定时循环
- 可随时停止定时功能

## 使用方法

1. 输入定时时间（分钟）
2. 点击"开始定时"
3. 时间到达后会自动锁屏5分钟
4. 锁屏期间可点击解锁按钮提前解除
5. 锁屏解除后会自动重新开始定时
6. 点击"停止定时"可彻底关闭功能
